# 关于

欢迎来到我的博客——

该博客通过 Astro 的 saicaca/fuwari 模板构建，并由 Github Page 托管，所以部分时候直连会有问题。fuwari 相对于其他模板有更多的基础样式，扁平化风格也适合朝各个方向更改。

与其说这是博客，其实更像一个实验室。我想通过博客的方式练习一下CSS内容，创建一些自认为不错的设计与功能。

如果你有任何想法，随便在issue或讨论里说。我的博客仓库没有规则，弄着玩的。
